template<typename InputT, typename DecorT, typename DataT>
class XMLB::detail::Parser< InputT, DecorT, DataT >

Класс, который управляет синтаксическим анализом входных данных.

Данный класс хранит наборы состояний-команд и данные для выполнения синтаксического анализа. Есть возможность создавать объекты данного класса со своими наборами состояний-команд и классом хранения данных.

Параметры шаблона
InputT- тип входящих символов символов
DecorT- тип декоратора
DataT- тип контроллера данных
