template<typename CharT>
struct XMLB::detail::Tag_data< CharT * >

Структура, которая хранит временные, результирующие данные синтаксического анализа входных XML даннхы

Это специализация для forward и выше итераторов, а также указателей. Данная структура используется при создания XML узлов. То есть данная структура является просто буфером.

Параметры шаблона
CharT- тип символов, которые будут храниться
