template<typename CharT>
class XMLB::Document< CharT >

XML документ с узлами

XML документ, который хранит в себе версию и кодировку документа. Также хранит root узел(тег) с его всеми дочерними узлами

Параметры шаблона
CharT- тип символов
Необходимо сделать:
Нужно подумать, нужно ли вообще хранить кодировку. Так же, возможно стоит приватно наследоваться от Node<CharT>.
